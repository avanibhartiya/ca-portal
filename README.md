# CA Portal

## 📌 Project Overview
This project is a web-based portal designed for Chartered Accountants (CAs) to access financial tools.

Currently, it includes:
- Secure Login System
- Capital Gains Calculator

Future enhancements:
- GST Calculator
- Income Tax Tools
- Report Generation

---
